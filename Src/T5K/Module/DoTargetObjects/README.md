## Do Target Objects

Parameters:
- UnitOfWork
    - Object/Entity to execute the action from
- ShotsFiredInBurst
    - Integer of shots in one burst  
- TimeBetweenBursts
    - Number to determine pause between bursts
- TargetArray
    - Array of targetable objects
- ExecutingGameLogic
    - An object for execution
- InfiniteAmmunition
    - Boolean
- RespectLineOfSight
    - Boolean
- LineOfSightLiteral
    - Constant
- LineOfSightThreshold
    - Constant
